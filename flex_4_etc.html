Properties for the flex container

flex-direction
row; row-reverse; column; column-reverse;
flex-wrap
nowrap; wrap; wrap-reverse;
align-items
flex-start;(위쪽을 기준으로 정렬 = vertical-align:top)
flex-end;(아래쪽을 기준으로 정렬 = vertical-align:bottom)
center;(wrap 박스의 수평 중앙를 기준으로 = vertical-align:middle)
baseline;(아이템 중에서 상하 길이가 긴것의 기준으로 수평정렬 = vertical-align:baseline)
stretch;(아이템들의 상하 길이를 wrap 박스의 높이에 맞게 통일)
justify-content
flex-start; (style = float:left;)
flex-end; (style = float:left; display:inline-block; + 부모태그에 text-align:right;)
space-between; (style = float:left; margin-right:10px; + 마지막 아이템 태그에 margin-right:0px)
center; (style = float:left; display:inline-block; + 부모태그에 text-alinn:center;)
space-around; (style = float:left; margin:0 10px;)
align-content (justify-content 스타일의 세로버전(행간의 간격 조절))

flex-end;
space-between;
center;
space-around;
